## How to publish with GitHub Pages

1. Create a public GitHub repository, for example:

```text
certifications
```

2. Upload these files to the repository.

3. Go to:

```text
Settings → Pages
```

4. Select:

```text
Deploy from a branch
```

5. Choose:

```text
Branch: main
Folder: /root
```

6. Save.

The site should become available at:

```text
https://YOUR-GITHUB-USERNAME.github.io/certifications/
```

Example:

```text
https://isaciotamayo.github.io/certifications/
```
